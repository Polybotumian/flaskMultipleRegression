Welcome to the Multiple Linear Regression Prediction App!

This web application allows you to perform predictions using a pre-trained multiple linear regression model or train your own model. Follow the steps below to get started:

Step 1: Upload Your Data for Prediction

Select the Model: From the dropdown menu, select the pre-trained regression model you want to use for predictions. Each model shows its performance metrics (MSE, R², and MAPE) to help you choose the best one.

Upload the CSV File: Click the "Choose File" button to upload the CSV file containing the input data you want to predict on. Ensure the CSV file is formatted correctly and matches the model's expected input format.

Submit the Form: After selecting the model and uploading your CSV file, click the "Predict" button to submit the form. The application will process your data and generate predictions.

Step 2: View Predictions

After submitting your data, the application will display the prediction results in a tabular format. You will see a table with two columns: "Index" and "Result", showing the index of the input data and the corresponding prediction result.

Step 3: Train Your Own Model

Upload Your Dataset: Click the "Choose File" button to upload the CSV file containing the training data. Ensure the CSV file is formatted correctly and contains both features and target variable.

Specify Model Name: Enter a name for the new model in the provided text field.

Submit the Form: Click the "Train" button to train a new model. The application will process your data, train a new model, and save it along with its performance metrics.

Example Workflow

  • Navigate to the Prediction Page: Go to the "Predict" section from the navigation bar.
  • Fill the Form for Prediction: Select a model from the dropdown menu. Each model is listed with its performance metrics (MSE, R², and MAPE) to help you choose the most suitable one. Upload your CSV file containing the data for which you need predictions. Click the "Predict" button.
  • Review Results: The predictions will be displayed on the page. Each prediction corresponds to a row from your input CSV file.
  • Fill the Form for Training: Upload a CSV file containing your training data and enter a name for the new model. Click the "Train" button to train and save the new model along with its performance metrics.

Tips for Using the App

  • CSV File Format: Make sure your CSV file does not contain headers if the model expects raw data without headers. The order of columns in the CSV should match the order expected by the model.
  • Model Selection: Choose the correct model that corresponds to the type of predictions you want to make. Different models may be trained on different datasets and have different input requirements. Pay attention to the performance metrics (MSE, R², and MAPE) to select the best model for your data.
  • Training New Models: When training a new model, ensure your dataset includes both the features and the target variable. Properly format your CSV file to match the expected input for training.
